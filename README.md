# 公開しやすい無料サイト

一番やりやすいのは **GitHub Pages** です。  
GitHub Pages は GitHub 上のリポジトリから HTML / CSS / JavaScript の静的サイトをそのまま公開できます。 citeturn0search4turn0search12

ゲームとして見せたいなら **itch.io** もかなり向いています。itch.io は HTML5 ゲームをブラウザで直接動かせます。 citeturn0search3turn0search15

Git 連携なしで置きやすさを重視するなら **Cloudflare Pages** も使いやすいです。Cloudflare Pages は Git 連携だけでなく、静的 HTML の直接アップロードにも対応しています。 citeturn0search5turn0search13

# 公開手順

## GitHub Pages
1. リポジトリを作る
2. このZIPの中身をアップロード
3. Settings → Pages で公開設定
4. 数十秒待つ

## itch.io
1. 新規プロジェクト作成
2. HTML5 / browser game として設定
3. ZIPをアップロード
4. ブラウザ実行を有効化

## Cloudflare Pages
1. 新規 Pages プロジェクト作成
2. このZIPの中身をアップロード、または GitHub 連携
3. そのまま公開
